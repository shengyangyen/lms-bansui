1. 註冊與登入

1.1 如何註冊新帳號

  1. 開啟瀏覽器，進入伴飛計畫網站網址（由主辦單位提供）。
  2. 在登入頁面點選「註冊」或類似連結，進入註冊頁面。
  3. 依序填寫：暱稱（顯示名稱）、真實姓名、電子郵件、密碼與確認密碼（密碼至少 6 個字元，兩次須一致）。
  4. 送出後若成功，畫面會提示：註冊成功！請檢查信箱驗證信，之後等待管理員審核。
  5. 請至信箱完成驗證（若有開啟此功能），並等待管理員核准後，方可使用完整登入功能。

【圖】註冊頁面完整表單

1.2 審核與登入

帳號在未核准前，可能無法登入或無法使用部分功能。核准後，使用註冊時的電子郵件與密碼登入。

1.3 登入步驟

  1. 進入網站登入頁。
  2. 輸入電子郵件與密碼。
  3. 點選登入。

【圖】登入頁面

1.4 登入後畫面卡住或很慢

網站架設於雲端，若一段時間沒人使用，伺服器可能進入休眠，第一次登入可能需等待約 1～2 分鐘，請勿重複狂按。若長時間無反應，可嘗試重新整理頁面（或按 Ctrl + Shift + R）後再試。

1.5 忘記密碼

若系統有提供「忘記密碼」功能，請依畫面指示操作。若無，請聯絡管理員或課程承辦人協助重設。

2. 儀表板概覽與課程

登入成功後，會進入個人首頁（儀表板）。頁面上方有一排分頁。

2.1 分頁說明

分頁名稱說明
飛行儀表板個人歡迎區、等級與經驗、徽章、學習動態等。
課程與作業列出已加入的課程，點入可進入該課程。
個人資訊管理修改登入密碼等。
訊息聯絡撰寫訊息給管理者（私訊）。
排行榜若管理員開放，可查看排行榜。

【圖】儀表板頂部分頁列

2.2 飛行儀表板內容

  • 歡迎訊息：顯示暱稱或名稱。
  • 飛行員等級／經驗值：依課程互動與作業等累積（規則以主辦單位說明為準）。
  • 徽章：達成條件後可能獲得。
  • 學習動態：近期學習相關紀錄摘要。

【圖】飛行儀表板主畫面

2.3 進入課程

  1. 點選分頁「課程與作業」。
  2. 在課程卡片列表中，點選欲進入的課程。
  3. 進入課程詳情頁，可看到課程標題與簡介。

【圖】課程列表　｜　【圖】單一課程頁頂部標題區

2.4 課程頁內的分頁

子分頁說明
教材瀏覽已上架教材（檔案、影片、連結等）。
作業查看指派給您的作業並前往繳交。
優秀作品觀摩老師標記的優秀作業（若課程有開放）。
討論課程留言與討論區。

【圖】課程頁—教材／作業／優秀作品／討論分頁

2.5 下載教材檔案

若教材為檔案類型且老師已設為可見，會出現「下載檔案」按鈕。點選後瀏覽器會下載；若失敗，請稍後再試或聯絡老師。

【圖】教材卡片與下載按鈕

2.6 影片與外部連結教材

YouTube 可能內嵌播放；雲端／網頁連結可能以「開啟雲端資料」「開啟網頁」等按鈕另開新分頁。

3. 作業內容與繳交

3.1 如何找到作業

  1. 進入課程頁，點「作業」分頁。
  2. 列表中僅顯示老師指定給您的作業。

【圖】課程—作業列表

3.2 進入繳交頁面

在作業卡片上點選「前往提交」或「前往作答」（依作業類型），進入繳交頁面。

【圖】作業卡片與提交按鈕

3.3 作業類型一：檔案上傳

  1. 閱讀作業標題與說明。
  2. 在「選擇要提交的檔案」區域，點選或拖放檔案。
  3. 支援常見格式（Word、PDF、圖片等）；單檔大小請遵守網站限制。
  4. 按「提交作業」完成上傳。

【圖】檔案上傳區與提交按鈕

3.4 作業類型二：填空題（線上作答）

  1. 依題號逐題填寫，標示 * 者為必答。
  2. 完成後按「提交答案」。
  3. 若老師有開放，可於提交狀態區匯出 PDF／Excel 留存。

【圖】填空題作答畫面

3.5 重複繳交與版本說明

系統說明為永久開放提交、可無限次修改（實際以老師公告為準）。尚未被批改時再次提交通常會覆蓋最新版本；已被批改後再提交會形成新版本，舊版本仍保留歷程。

【圖】提交頁下方小提示區塊

3.6 右側「提交狀態」

顯示最新提交時間、待批改或已批改；檔案作業可「下載已提交的檔案」；已批改時可進入「查看完整批改記錄」。

【圖】提交狀態側欄

4. 作業成果與回饋

4.1 在繳交頁快速查看批改摘要

老師已批改時，提交狀態區會顯示評分（建議需調整／合格／優秀）與部分意見。點「查看完整批改記錄」可開啟完整頁面。

4.2 批改記錄頁面

標題為「作業批改歷史」／「批改記錄」。每筆含評分、時間、批改人、批改意見；若有圖片或檔案，可下載批改檔案。

【圖】批改歷史列表　｜　【圖】單筆批改含下載按鈕

4.3 評分意義（介面顯示）

評分一般意涵（以老師說明為準）
建議需調整需依意見修改後可能再提交。
合格達到基本要求。
優秀表現突出。

7. 常見問題與注意事項

狀況建議作法
登入一直轉圈先等 1～2 分鐘；仍不行則重新整理或無痕模式。
帳號或密碼錯誤確認信箱；忘記密碼聯絡管理員。
看不到課程確認帳號已核准且已被加入課程。
看不到作業作業可能僅指定給部分學員，洽老師。
教材無法下載洽老師或稍後再試。
網站突然無法使用可能維護或雲端暫停，稍後再試或洽主辦單位。
