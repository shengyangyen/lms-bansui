# 伴飛計畫 - 設計系統使用指南

## 概覽

本項目已升級至「藝術簡約風格」，基於以下核心價值：
- 🎨 **簡約**: 最少化設計元素，避免視覺過載
- ✈️ **飛航主題**: 天空、航道、成長的隱喻
- 🎯 **易用性**: 清晰的視覺層次和互動反饋

---

## 色彩系統

### 主色板
```
主色：#609ea3 (協會藍綠)
淡色背景：#e8f2f4
深色：#4a7d82
```

### 使用方式

```jsx
// Tailwind CSS 類名
<button className="text-primary">按鈕文字</button>
<div className="bg-primary">深色背景</div>
<div className="bg-primary-light">淡色背景</div>
```

### 全局色彩轉換

舊代碼中的藍色配色會自動轉換：
```
.bg-blue-500 → 協會主色
.text-blue-600 → 協會主色深色
.border-blue-500 → 協會主色邊框
```

## 字型系統

### 類別

**標題 (Heading)**
```jsx
<h1 className="font-heading">標題</h1>
<!-- Heiti TC / 黑體 / PingFang SC -->
```

**內文 (Body)**
```jsx
<p className="font-body">內文</p>
<!-- Ming SC / 明體 / Songti SC / 宋體 -->
```

### 字型大小
- xs: 12px
- sm: 14px (標籤、說明)
- base: 16px (預設)
- lg: 18px (小標題)
- xl: 20px (大標題)

## 可重用 UI 組件

### 1. 按鈕

```jsx
// 主要按鈕 (外框設計)
<button className="btn-primary">
  主要按鈕
</button>

// 次要按鈕
<button className="btn-secondary">
  次要按鈕
</button>

// 危險操作
<button className="btn-danger">
  刪除
</button>

// 成功操作
<button className="btn-success">
  確認
</button>
```

**樣式特點**
- 外框設計，無實心背景
- Hover: 背景變淡，邊框加深
- Focus: 明確的視覺反饋

### 2. 卡片

```jsx
<div className="card">
  <h2 className="card-header">卡片標題</h2>
  <div className="card-body">
    卡片內容
  </div>
</div>
```

**特點**
- 輕微陰影 (shadow-base)
- 圓角邊框
- 白色背景
- Hover: 陰影增強

### 3. 分頁標籤

```jsx
import { TabsContainer, Tab } from '../components/Tabs';

<TabsContainer>
  <Tab 
    label="標籤 1" 
    isActive={activeTab === 'tab1'}
    onClick={() => setActiveTab('tab1')}
  />
  <Tab 
    label="標籤 2" 
    isActive={activeTab === 'tab2'}
    onClick={() => setActiveTab('tab2')}
  />
</TabsContainer>
```

**設計**
- 下劃線表示選中
- 選中時有淡色背景
- 未選中可 Hover 預覽

### 4. 表單輸入

```jsx
<div>
  <label className="input-label">
    標籤文字
  </label>
  <input className="input-field" />
</div>
```

### 5. 提示訊息

```jsx
// 錯誤
<div className="alert-error">
  錯誤訊息
</div>

// 成功
<div className="alert-success">
  成功訊息
</div>

// 警告
<div className="alert-warning">
  警告訊息
</div>

// 資訊
<div className="alert-info">
  資訊訊息
</div>
```

## 最佳實踐

### 1. 使用語義化的 CSS 類

✅ **好**
```jsx
<button className="btn-primary">登入</button>
<div className="card">...</div>
```

❌ **不好**
```jsx
<button className="px-4 py-2 bg-blue-500 text-white">登入</button>
<div className="border rounded shadow p-6">...</div>
```

### 2. 保持視覺一致性

- 同一類型的按鈕使用相同的樣式
- 卡片間距統一 (`gap-6`, `mb-8`)
- 文字大小階層清晰 (heading > lg > base > sm)

### 3. 避免過度設計

- 不要使用超過 2-3 種強調色
- 保留足夠的留白
- Icon 最小化使用

### 4. 響應式設計

```jsx
// 使用 Tailwind 的響應式前綴
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  ...
</div>
```

## 常見模式

### 內容卡片網格

```jsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  {items.map(item => (
    <div key={item.id} className="card hover:shadow-lg transition">
      <h3 className="card-header">{item.title}</h3>
      <p className="card-body">{item.description}</p>
    </div>
  ))}
</div>
```

### 表單容器

```jsx
<div className="card">
  <h2 className="card-header">表單標題</h2>
  <form onSubmit={handleSubmit} className="space-y-4">
    <div>
      <label className="input-label">欄位 1</label>
      <input className="input-field" />
    </div>
    <button className="btn-primary">提交</button>
  </form>
</div>
```


