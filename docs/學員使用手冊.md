## 1. 註冊與登入

### 1.1 如何註冊新帳號

1. 開啟瀏覽器，進入伴飛計畫網站網址（由主辦單位提供）。
2. 在登入頁面點選「註冊」或類似連結，進入註冊頁面。
3. 依序填寫：
   - **暱稱（顯示名稱）**：在系統內顯示給他人看的名字。
   - **真實姓名**：供管理與審核使用。
   - **電子郵件**：請填寫常用且可收信的信箱。
   - **密碼**與**確認密碼**：至少 **6 個字元**，兩次須一致。
4. 送出表單後，若成功，畫面會提示：**註冊成功！請檢查信箱驗證信，之後等待管理員審核。**
5. 請至信箱完成驗證（若有開啟此功能），並**等待管理員核准**後，方可使用完整登入功能。

【圖：註冊頁面完整表單】

### 1.2 審核與登入

- 帳號在**未核准**前，可能無法登入或無法使用部分功能。
- 核准後，使用註冊時的**電子郵件**與**密碼**登入。

### 1.3 登入步驟

1. 進入網站登入頁。
2. 輸入**電子郵件**與**密碼**。
3. 點選登入。

【圖：登入頁面】

### 1.4 登入後畫面卡住或很慢

- 網站架設於雲端，若**一段時間沒人使用**，伺服器可能進入休眠，**第一次登入**可能需等待約 **1～2 分鐘**，請勿重複狂按。
- 若長時間無反應，可嘗試重新整理頁面（或按 `Ctrl + Shift + R` 強制重新載入）後再試。

### 1.5 忘記密碼

- 若系統有提供「忘記密碼」功能，請依畫面指示操作。
- 若無，請聯絡**管理員**或課程承辦人協助重設。

## 2. 儀表板概覽與課程

登入成功後，會進入**個人首頁（儀表板）**。頁面上方有一排**分頁**，可依需要切換。

### 2.1 分頁說明

| 分頁名稱       | 說明 |
|----------------|------|
| **飛行儀表板** | 個人歡迎區、等級與經驗、徽章、學習動態等。 |
| **課程與作業** | 列出您已加入的課程，點入可進入該課程。 |
| **個人資訊管理** | 修改登入密碼等。 |
| **訊息聯絡**   | 撰寫訊息給管理者（私訊）。 |
| **排行榜**     | 若管理員開放，會出現此分頁，可查看排行榜。 |

【圖：儀表板頂部分頁列】

### 2.2 飛行儀表板內容

- **歡迎訊息**：顯示您的暱稱或名稱。
- **飛行員等級／經驗值**：依課程互動與作業等累積（實際規則以主辦單位說明為準）。
- **徽章**：達成條件後可能獲得徽章展示。
- **學習動態**：近期與學習相關的紀錄摘要。

【圖：飛行儀表板主畫面】

### 2.3 進入課程

1. 點選分頁 **「課程與作業」**。
2. 在課程卡片列表中，點選欲進入的課程（例如「進入課程」）。
3. 進入**課程詳情頁**，可看到課程標題與簡介。

【圖：課程列表】  
【圖：單一課程頁頂部標題區】

### 2.4 課程頁內的分頁（教材／作業／優秀作品／討論）

在課程頁中，另有四個子分頁：

| 子分頁     | 說明 |
|------------|------|
| **教材**   | 瀏覽已上架的教材（檔案、影片連結、雲端連結等）。 |
| **作業**   | 查看指派給您的作業並前往繳交。 |
| **優秀作品** | 觀摩老師標記的優秀作業（若課程有開放）。 |
| **討論**   | 課程留言與討論區（見第 5 章）。 |

【圖：課程頁—教材／作業／優秀作品／討論分頁】

### 2.5 下載教材檔案

- 若教材為**檔案類型**，且老師已設為可見，會出現 **「下載檔案」** 按鈕。
- 點選後瀏覽器會下載檔案；若失敗，請稍後再試或聯絡老師。

【圖：教材卡片與下載按鈕】

### 2.6 影片與外部連結教材

- **YouTube**：部分教材會直接內嵌播放。
- **雲端／網頁連結**：可能以「開啟雲端資料」「開啟網頁」等按鈕另開新分頁。

## 3. 作業內容與繳交

### 3.1 如何找到作業

1. 進入課程頁，點 **「作業」** 分頁。
2. 列表中僅會顯示**老師指定給您**的作業（若您看不到某份作業，可能是尚未指定，請洽老師）。

【圖：課程—作業列表】

### 3.2 進入繳交頁面

- 在作業卡片上點選 **「前往提交」** 或 **「前往作答」**（依作業類型顯示不同文字），進入繳交頁面。

【圖：作業卡片與提交按鈕】

### 3.3 作業類型一：檔案上傳

1. 閱讀作業標題與說明。
2. 在 **「選擇要提交的檔案」** 區域，點選或拖放檔案。
3. 支援常見格式（如 Word、PDF、圖片等）；單檔大小請遵守網站限制（若有提示）。
4. 按 **「提交作業」** 完成上傳。

【圖：檔案上傳區與提交按鈕】

### 3.4 作業類型二：填空題（線上作答）

1. 依題號逐題填寫，**標示 * 者為必答**。
2. 填寫完成後按 **「提交答案」**。
3. 若老師有開放，您可能可以**匯出 PDF／Excel** 留存自己的答案（於提交狀態區查看連結）。

【圖：填空題作答畫面】

### 3.5 重複繳交與版本說明

- 系統說明：**永久開放提交，可無限次修改**（實際截止與規則以老師公告為準）。
- **尚未被批改**：再次提交通常會**覆蓋**目前最新版本。
- **已被批改**：再提交會形成**新版本**，舊版本仍會保留供老師與您查閱歷程。

【圖：提交頁下方小提示區塊】

### 3.6 右側「提交狀態」

- 顯示**最新提交時間**、是否**待批改**或**已批改**。
- 檔案作業可 **「下載已提交的檔案」** 確認上傳內容。
- 已批改時可從此處進入 **「查看完整批改記錄」**（詳見第 4 章）。

【圖：提交狀態側欄】

## 4. 作業成果與回饋

### 4.1 在繳交頁快速查看批改摘要

- 若老師已批改，在「提交狀態」區會顯示 **評分**（如：建議需調整／合格／優秀）與部分**批改意見**。
- 點 **「查看完整批改記錄」** 可開啟完整頁面。

### 4.2 批改記錄頁面

- 標題為 **「作業批改歷史」**／**「批改記錄」**。
- 每一筆記錄包含：**評分**、**時間**、**批改人**、**批改意見**（文字）。
- 若老師有上傳**圖片或檔案**，會顯示預覽或 **「下載批改檔案」** 按鈕。

【圖：批改歷史列表】  
【圖：單筆批改含下載按鈕】

### 4.3 評分意義（介面顯示）

| 評分       | 一般意涵（實際以老師說明為準） |
|------------|--------------------------------|
| 建議需調整 | 需依意見修改後可能再提交。     |
| 合格       | 達到基本要求。                 |
| 優秀       | 表現突出。                     |

### 4.4 返回上一頁

- 使用頁面上方 **「← 返回」** 可回到前一頁。

## 5. 討論

### 5.1 進入討論區

1. 在課程頁點選 **「討論」** 分頁。
2. 可瀏覽他人留言與老師回覆。

【圖：課程討論分頁】

### 5.2 發表留言

1. 在輸入框撰寫內容。
2. 送出後會顯示在討論串中。
3. 請遵守網路禮儀與課程規範；惡意或違規內容可能由管理員處理。

### 5.3 回覆留言

- 若介面支援對某則留言 **回覆**，可點選回覆後在子串中繼續討論。

【圖：留言輸入與列表】

## 6. 其他功能

### 6.1 個人資訊管理（修改密碼）

1. 在儀表板點 **「個人資訊管理」**。
2. 填寫 **目前密碼**、**新密碼**（至少 6 字）、**確認新密碼**。
3. 送出後若成功，會提示密碼已更新。

【圖：修改密碼表單】

### 6.2 訊息聯絡（私訊管理者）

1. 點 **「訊息聯絡」**。
2. 輸入想傳給管理者的內容並送出。
3. 下方可查看**已發送訊息**與管理者**回覆**（若有）。

【圖：訊息聯絡頁面】

### 6.3 排行榜

- 若頂部有 **「排行榜」** 分頁，或飛行儀表板出現「前往排行榜」，可進入查看排名（規則由主辦單位訂定）。

【圖：排行榜頁面】

### 6.4 優秀作品

- 在課程 **「優秀作品」** 分頁可觀摩同儕或範例作業（權限依角色與老師設定而定）。

【圖：優秀作品列表】

### 6.5 登出

- 點選畫面上的 **「登出」**，可安全結束本次使用，特別是在公用電腦上務必登出。

【圖：登出按鈕位置】

## 7. 常見問題與注意事項

| 狀況 | 建議作法 |
|------|----------|
| 登入一直轉圈 | 先等待 1～2 分鐘；仍不行則重新整理或換瀏覽器／無痕模式。 |
| 提示帳號或密碼錯誤 | 確認信箱無誤；若確定忘記密碼，聯絡管理員。 |
| 看不到課程 | 確認帳號已核准，且已被加入該課程。 |
| 看不到作業 | 作業可能僅指定給部分學員，請洽老師。 |
| 教材無法下載 | 可能檔案尚未上線或連線問題，請洽老師。 |
| 網站突然無法使用 | 可能是雲端服務維護或資料庫暫停，稍後再試或洽主辦單位。 |
