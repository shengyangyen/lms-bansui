# LMS 角色權限設定表

填寫完成後可作為實作依據。請依實際需求勾選或修改。

---

## 一、角色總覽

| 角色代碼 | 顯示名稱 | 說明 |
|---------|----------|------|
| admin | 管理者 | 最高權限 |
| instructor | 導師 | 教材與作業完整管理，無獎章/經驗值權限 |
| flight_instructor | 飛行導師 | 觀看教材作業、參與討論，不累計經驗值與獎章 |
| student | 學員 | 觀看指定內容、繳交作業、參與討論 |
| study_buddy | 共學之友 | 觀看特定教材、優秀作品、參與討論，無法閱覽作業 |
