## 🔍 實戰發現的問題 & 解決方案

### 問題 1：速率限制過嚴
```
現象：快速操作時頻繁收到「請求過於頻繁」錯誤
根因：設置為 15 分鐘 100 個請求（太嚴格）
解決：
  - localhost 完全跳過限制
  - 生產環境改為 15 分鐘 1000 個請求
  - 檢查：NODE_ENV 環境變數
```

### 問題 2：中文檔名亂碼
```
現象：下載上傳的中文檔名檔案時變成亂碼（如「測試.doc」→「èªç¶.doc」）
根因：Multer 在處理檔案時用 latin1 而非 UTF-8 解碼
解決：
  1. 後端：multer.fileFilter 加入編碼修正
     file.originalname = Buffer.from(file.originalname, 'latin1').toString('utf8')
  
  2. 前端：改用 Blob API 下載，不依賴 HTTP header
     - 舊方式：link.href = URL + Content-Disposition header
     - 新方式：const blob = await fetch(); link.download = filename

  修復位置：
  - backend/src/index.js (L71-82)
  - backend/src/assignments.js (L20-31)
  - frontend/src/pages/SubmitAssignment.jsx (L261-281, L395-417)
  - frontend/src/pages/CourseDetail.jsx (L147-165)
```

### 問題 3：新角色無法加入課程
```
現象：「伴飛學員」和「共學之友」帳號無法添加到課程
根因：API 只查詢 user_role='student' 的用戶
解決：改用 IN 查詢多個角色
  .in('user_role', ['student', 'trainee', 'study_buddy'])
  
修復位置：backend/src/index.js (L492-505)
```
