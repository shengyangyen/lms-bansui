```css

```

```jsx

<TopNav title="..." showAdminLink={true} />

// Tabs - 分頁標籤

```

## 🚀 使用新設計系統

### 最常用的類名

```jsx
// 色彩
.text-primary           // 主色文字
.bg-primary-light       // 淡色背景
.border-primary         // 主色邊框

// 字型
.font-heading           // 標題
.font-body              // 內文

// 組件
.card                   // 卡片容器
.btn-primary            // 主要按鈕
.alert-success          // 成功提示

// 佈局
.page-container         // 頁面容器
.input-field            // 表單輸入
```

### 完整範例

```jsx
import TopNav from '../components/TopNav';

export function MyPage() {
  return (
    <div className="page-container min-h-screen">
      {/* 導航 */}
      <TopNav title="我的頁面" showAdminLink={true} />
      
      {/* 內容 */}
      <div className="max-w-7xl mx-auto py-12 px-4">
        <div className="card">
          <h2 className="font-heading text-2xl font-bold">標題</h2>
          <p className="font-body text-gray-700">內文</p>
          <button className="btn-primary">按鈕</button>
        </div>
      </div>
    </div>
  );
}
```

## ❓ 常見問題

### Q: 新設計是什麼風格？
A: **藝術簡約風格** - 最少化設計元素，避免視覺過載，融入飛航訓練主題。

### Q: 舊頁面會自動變成新色彩嗎？
A: **是的！** 通過全局色彩轉換系統，所有舊的藍色配色自動變成新主色。

### Q: 需要手動升級每個頁面嗎？
A: **不完全需要。** 色彩已自動轉換，但為了達到最佳視覺效果，推薦逐步升級高優先級頁面。

### Q: 如何在新頁面中使用新設計系統？
